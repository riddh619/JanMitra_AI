## 📌 Overview

**JanMitra AI** is an **AI-powered constituency planning assistant** designed to help Members of Parliament (MPs) make informed development decisions.

Instead of manually reviewing hundreds of citizen suggestions, JanMitra AI automatically analyzes multilingual citizen feedback, combines it with public infrastructure data, and generates **transparent, evidence-backed priority rankings** for development works.

Built for the **Build with AI - Code for Communities Hackathon**, the platform demonstrates how **Generative AI + Agentic AI** can bridge the gap between citizens and decision-makers.

---

## 🚀 Problem Statement

Government representatives receive numerous citizen suggestions regarding:

- 🏥 Healthcare
- 🚰 Water Supply
- 🛣️ Roads
- 🎓 Education
- ⚡ Electricity
- 🌱 Agriculture

Manually reviewing, categorizing, and prioritizing these requests is time-consuming and often lacks data-driven justification.

JanMitra AI transforms these unstructured citizen voices into actionable development insights.

### 🤖 Agentic AI Pipeline

Instead of a single AI prompt, JanMitra AI follows an intelligent workflow:

```
Citizen Suggestion
        │
        ▼
Classification Agent
        │
        ▼
Dataset Retrieval Agent
        │
        ▼
Impact Analysis Agent
        │
        ▼
Priority Scoring Agent
        │
        ▼
Recommendation Generator
        │
        ▼
MP Dashboard
```

Each stage contributes towards producing transparent and explainable recommendations.
